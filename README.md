

# Tool_N
This tool is a general purpose terminal-based tool that provides miscallenous and/or unnecessary things.

# Getting Started
## Dependencies
 - Windows 10-11

## Installation
 1. Go to the releases page.
 2. Install the correct package depending on your operating system.
 3. Add the executable to PATH.

## Usage
To use the program on every directory, you may need to put it in PATH.
Some basic flags to check if the program works are:
```
Tool_N --version
Tool_N -v
Tool_N --help
Tool_N -h
```
